Version 1.0.0 Notes

Established QR Code Functions

Established QR Code functions, generation, and analytics

  • Established New QR Code Module, QR Code List, and Create QR Code page
  • QR Code list containing Discount Code, Location, Sales Analytics Data, and Redirect Analytics Data
  • Create QR Code page contains QR Code Type, Location, Discount Title(From Shopify), Redirect URL field, and Generation Feature to create the QR Code for Download
  • Added Inactive and Delete actions for QR Codes. Delete is only optional if no Analytic Data is present
  • Locations Information updated to include the Full URL field used for redirection when location and shopify are selected in the generation process

End of Day Reporting Updates

Updates to the End of Day entry functions for a Minor Calculation error for Multiple Payment Charge Card

Order Details Update

Updated Orders Detail page to fetch all Oncard numbers for that order and display on page. If Card had been clear it will display "Card History Cleared" and show a Balance_id when hovering over text.

Reporting Update

Updated all reporting features

  • Reports can now be Downloaded and Printed after generated
  • Updated reports to display selected Locations and Devies at the bottom of report

Minor Bug Fixes

Minor update to disable the Sidebar if a Pop-up window is present
