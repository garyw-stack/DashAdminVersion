Version 0.0.2 Notes

Order List and Details updates

Update to Order List and Details page Visuals and functions

  • Visual updates to Rename columns in Orders List to align with required fields
  • Updated Columns to allow for filters ascending and decending of rows
  • Updated Order Details page to display Oncard number as a Drilldown link

Shopify Sales Updates

Visual and Functional updates to Shopify Sales

  • Update the language from "Shift Start" and "Shift End" to "Open" and "Close"
  • Added a "Change Due" pop-up during Cash Transactions, if change is due then Window requires manual dismissal and if no change due will auto Dismiss

Location Page Update

Update to Location Page to add a new Value field and toggle to add a threshold amount for Balance Builder and Create/Split Card function to bypass the hardcoded default of 60 points

Minor Bug Fix

Minor bug fix to round decimal values during the Create/Split Card function during Sales process
