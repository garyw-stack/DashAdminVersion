Version 0.0.5 Notes

Established Sales Report for End of Month

Established Sales Reports and Selection Functions for End of Month

  • Create new Module for End of Month with Dropdown selections for Locations, Device Type(s), Device(s), Date Range, and Report Type
  • Generate Report layout based on the Report Type selected; Complete or Summary
  • Established Print and Download options to EoD report

Update to Shift Manager Page

Updated Visual indictor on the Shift Manager page based on certain factors

  • Shift Status visuals based on shift status
  • Register reported sales visual if drawer value entered is shorter than reported sales

Established Transaction Order

Established Transaction lookup features and Receipt updates

  • Created new module for Transaction Lookup based on Order Number
  • Updated Reciept layout to include a Transaction Barcode at base of receipt that is same as Order number

Order Details Page updates

Visual and Functional updates to the Order Details page

  • Added new section on the Details page to display Charge Card information
  • Added Reprint Receipt button
  • Added the Refund button for future function
  • Visual update for Capitalization of Payment details

Bug Fixes

Minor Bug fixes to Point of Sales system

  • Minor bug fix to reprinting Transaction receipts when multiple payment option was used
