Version 1.0.3 Notes

Point of Sale Functionality Updates

Update to the Point of Sales Functionality and Workflow features

  • Updated the Point of Sale to keep Current Shift active until an authorized user Ends the current shift
  • Updated Point of Sale to only be available for use if the device type is Point of Sale
  • Updated to keep retain current page when swapping between users with "Quick Swap" function

Update to Order Page

Update "Transaction By" field in the Orders Page to display the User that was logged in during transaction, originally set for user that started shift

Roles and Permissions Update

Updates to the Roles and Premission Features

  • Enable Point of Sale Premission based logic for testing
  • Minor update to Transaction lookup premissions for Cashier use, originally Manager only function

Point of Sale to Table Update

Update Credit Balance to display correctly in Table for Transaction history when using Point of Sale for the transaction
